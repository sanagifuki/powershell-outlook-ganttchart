# 仕様メモ

## データファイル

- `schedules.json`: Outlook同期データのキャッシュ。
- `logs.json`: 作業ログ。
- `error.log`: Outlook同期エラーなど。

これらは `.gitignore` で除外する。

## タイトル記号

- `✕`: 絶対期限。
- `◆`: 推奨期限。
- `◇`: 目安期限。
- `▶`: 予定日。
- `★`: 参照用。

## ステータス

Outlookカテゴリーを優先する。

- `完了`: ステータスを完了にする。
- `廃止`: ステータスを廃棄にする。
- `★` かつ未完了/未廃棄: 表示。
- それ以外: 未着手。

## ガント表示

現時点では既存コードの仕様を維持する。詳細な判定は今後 `Domain/` 側へ切り出して整理する。
